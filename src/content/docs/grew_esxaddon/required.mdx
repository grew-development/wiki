In this section, you’ll find an overview of the required and optional scripts necessary for the *Grew esxAddon* to function properly.
<Aside type="tip">
  To get started, ensure you have installed the **required scripts**.<br/>
  Once those are functioning correctly, consider adding the **optional scripts** to enhance your server’s capabilities further.
</Aside>

## Required Scripts:
- ### <a href="https://github.com/esx-framework/esx-legacy" target="_blank" style="text-decoration: none;">es_extended Legacy</a>
  - **Description**: The ESX Legacy script forms the backbone for the *Grew esxAddon*.  
  - **Version**: Compatible from version 1.9.0 onwards.


## Optional Scripts:
- ###  <a href="https://wiki.grew-development.de/grew_versionchecker/readme" target="_blank" style="text-decoration: none;">Grew Versionchecker</a>
  - **Description**: This script checks for updates on all *Grew Development Network* scripts, notifying the server owner when new versions are available.
  - **Version**: Compatible from version 1.2.0 onwards.
