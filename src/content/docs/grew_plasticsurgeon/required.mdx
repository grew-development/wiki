In this section, you’ll find an overview of the required and optional scripts necessary for the *Grew Plasticsurgeon* to function properly.
<Aside type="tip">
  To get started, ensure you have installed the **required scripts**.<br/>
  Once those are functioning correctly, consider adding the **optional scripts** to enhance your server’s capabilities further.
</Aside>

## Required Scripts:
### ESX Framework
- #### <a href="https://github.com/esx-framework/esx_core" target="_blank" style="text-decoration: none;">es_extended Legacy</a>
  - **Description**: The ESX Legacy script forms the backbone for the *Grew Plastisurgeon*.  
  - **Version**: Compatible from version 1.9.0 onwards.
  
- #### <a href="https://github.com/esx-framework/esx_core" target="_blank" style="text-decoration: none;">esx_skin</a>
  - **Description**: The ESX Skin script handles the seamless saving of player appearance changes in *Grew Plasticsurgeon*.  
  - **Version**: Compatible from version 1.9.0 onwards.

### QB Framework
- #### <a href="https://github.com/qbcore-framework/qb-core" target="_blank" style="text-decoration: none;">qb-core</a>
  - **Description**: The QB-Core script forms the backbone for the *Grew Plastisurgeon*.  
  - **Version**: Compatible from version 1.3.0 onwards.
  
- #### <a href="https://github.com/qbcore-framework/qb-clothing" target="_blank" style="text-decoration: none;">qb-clothing</a>
  - **Description**: The QB-Clothing script handles the seamless saving of player appearance changes in *Grew Plasticsurgeon*.  
  - **Version**: Compatible from version 1.2.0 onwards.

## Optional Scripts:
- ####  <a href="https://wiki.grew-solutions.de/grew_versionchecker/readme" target="_blank" style="text-decoration: none;">Grew Versionchecker</a>
  - **Description**: This script checks for updates on all *Grew Solutions Network* scripts, notifying the server owner when new versions are available.
  - **Version**: Compatible from version 1.2.0 onwards.
