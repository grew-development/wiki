In diesem Abschnitt findest du eine Übersicht über die erforderlichen und optionalen Scripts, die notwendig sind, damit *Grew esxAddon* ordnungsgemäß funktioniert.
<Aside type="tip">
  Um zu beginnen, stelle sicher, dass du die **erforderlichen Scripts** installiert hast.<br/>
  Sobald diese korrekt funktionieren, kannst du die **optionalen Scripts** hinzufügen, um die Möglichkeiten deines Servers weiter zu erweitern.
</Aside>

## Erforderliche Scripts:
- ### <a href="https://github.com/esx-framework/esx-legacy" target="_blank" style="text-decoration: none;">es_extended Legacy</a>
  - **Beschreibung**: Das ESX Legacy Script bildet die Grundlage für das *Grew Plasticsurgeon*.  
  - **Version**: Kompatibel ab Version 1.9.0.

- ### <a href="https://github.com/esx-framework/esx-legacy" target="_blank" style="text-decoration: none;">esx_skin</a>
  - **Beschreibung**: Das ESX Skin Script sorgt für die reibungslose Speicherung von Änderungen am Erscheinungsbild der Spieler in *Grew Plasticsurgeon*.  
  - **Version**: Kompatibel ab Version 1.9.0.

## Optionale Scripts:
- ###  <a href="https://wiki.grew-solutions.de/grew_versionchecker/readme" target="_blank" style="text-decoration: none;">Grew Versionchecker</a>
  - **Beschreibung**: Dieses Script prüft auf Updates aller *Grew Solutions Network*-Scripts und informiert den Serverbesitzer, wenn neue Versionen verfügbar sind.
  - **Version**: Kompatibel ab Version 1.2.0.
