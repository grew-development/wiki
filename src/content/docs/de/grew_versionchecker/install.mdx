Diese Anleitung zeigt dir, wie du das Grew Versionchecker-Script auf deinem Server installierst und konfigurierst.<br/>
Der Grew Versionchecker unterstützt dich dabei, deine Skripte automatisch auf dem aktuellen Stand zu halten und erleichtert die Verwaltung von Updates.<br/>
Folge den unten stehenden Schritten, um das Script herunterzuladen, hochzuladen und nach deinen Wünschen anzupassen.<br/>
Stelle sicher, dass du die Installationsschritte genau befolgst, um das Script erfolgreich zu implementieren.<br/>
<br/>

<Steps>
  1. Kaufe das Script in meinem Store.
  
  2. Lade Grew_Versionchecker von <a href="https://keymaster.fivem.net/" target="_blank">`https://keymaster.fivem.net/`</a> herunter.

  3. Lade den Ordner `grew_versionchecker` hoch.

  4. Füge `ensure grew_versionchecker` in deine server.cfg ein.

  5. Bearbeite die Config-Datei nach deinen persönlichen Wünschen.<br/>
      Für Fragen zur Config-Datei siehe <a href="./config" target="_self">config</a>.

  6. Starte deinen Server neu und genieße das Script.
</Steps>
