Der Grew Versionchecker ist ein unverzichtbares Script für das Grew Solutions Network.<br/>
Es dient dazu, andere Scripte im Netzwerk auf ihre aktuelle Version zu überprüfen und sicherzustellen, dass alle Scripte auf dem neuesten Stand sind.<br/>
Mit diesem Script bist du immer einen Schritt voraus und ersparst dir die Mühe, ständig manuell nach Updates suchen zu müssen!

## Funktionen
* Versionüberprüfung: Das Script prüft, ob alle im Grew Solutions Network laufenden Scripte aktuell sind.

* Benachrichtigung in der Serverkonsole: Falls eine veraltete Version eines Scripts gefunden wird,<br/>
gibt der Versionchecker eine entsprechende Nachricht in der Serverkonsole aus.

* Vermeidung von Versionskonflikten: Durch die regelmäßige Überprüfung stellt der Versionchecker sicher,<br/>
dass keine älteren Versionen von Scripten im Netzwerk verwendet werden, was mögliche Kompatibilitätsprobleme oder Bugs verhindert.

## Wichtig
Dieses Script muss installiert sein, damit die Überprüfung der Versionen der Scripte im Grew Solutions Network stattfinden kann.<br/>
Ohne den Grew Versionchecker wird keine Versionenprüfung durchgeführt und es besteht die Gefahr, dass veraltete Scripte unbemerkt im Netzwerk verbleiben.
