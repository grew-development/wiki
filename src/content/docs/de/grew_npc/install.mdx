Diese Anleitung zeigt dir, wie du das Grew NPC-Skript auf deinem Server installierst und konfigurierst.<br/>
Folge den unten stehenden Schritten, um das Skript herunterzuladen, hochzuladen und anzupassen, damit es den Anforderungen deines Servers entspricht.<br/>
Achte darauf, die Installationsschritte genau zu befolgen, um das Skript erfolgreich zu integrieren und sein volles Potenzial für erweiterte Serverfunktionen freizuschalten.<br/>
<br/>

<Steps>
  0. Stelle sicher, dass alle erforderlichen Skripte installiert sind, bevor du fortfährst.<br/>
     Installiere bei Bedarf optionale Skripte, um die Funktionalität zu erweitern.

  1. Kaufe das Skript in meinem Store.

  2. Lade Grew NPC von <a href="https://keymaster.fivem.net/" target="_blank">`https://keymaster.fivem.net/`</a> herunter.

  3. Lade den Ordner `grew_npc` hoch.

  4. Füge `ensure grew_npc` in deine server.cfg ein.

  5. Bearbeite die Konfigurationsdatei nach deinen persönlichen Wünschen.<br/>
     Bei Fragen zur Konfigurationsdatei siehe <a href="/grew_npc/config" target="_self">config</a>.

  6. Starte deinen Server neu und viel Spaß
</Steps>
