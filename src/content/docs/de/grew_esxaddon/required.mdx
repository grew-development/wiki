In diesem Abschnitt findest du eine Übersicht der erforderlichen und optionalen Scripts, die für die korrekte Funktion des *Grew esxAddon* Script notwendig sind.
<Aside type="tip">
  Um loszulegen, stelle sicher, dass du die **erforderlichen Scripts** installiert hast.<br/>
  Sobald diese einwandfrei funktionieren, kannst du die **optionalen Scripts** hinzufügen, um die Möglichkeiten deines Servers weiter auszubauen.
</Aside>


## Erforderliche Scripts:
- ### <a href="https://github.com/esx-framework/esx-legacy" target="_blank" style="text-decoration: none;">es_extended Legacy</a>
  - **Beschreibung**: Das ESX Legacy Script bildet das Fundament für das *Grew esxAddon*.  
  - **Version**: Kompatibel ab Version 1.9.0.


## Optionale Scripts:
- ###  <a href="https://wiki.grew-development.de/grew_versionchecker/readme" target="_blank" style="text-decoration: none;">Grew Versionchecker</a>
  - **Beschreibung**: Dieses Script überprüft Updates für alle *Grew Development Network* Scripts und benachrichtigt den Serverinhaber, wenn neue Versionen verfügbar sind.
  - **Version**: Kompatibel ab Version 1.2.0.
