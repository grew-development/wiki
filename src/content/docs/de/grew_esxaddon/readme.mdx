Dieses Addon erweitert ESX ab Version 1.9.0. <br/>
Hier werden neue Befehle erstellt, die einfach ein Inventar der ESX-Befehle darstellen. <br/>
Zusätzlich erstellt dieses Script einige Exporte, die vielseitig für die Programmierung oder meine Scripts genutzt werden können.

## Funktionen
* Erweiterung von es_Extended: Das **esxAddon**-Script erweitert das **es_Extended**-Modul des **EWX-Frameworks** und<br/>
bietet zusätzliche Funktionen zur Entwicklung von FiveM-Servern.

* Unterstützung für syncCallbacks: Mit **esxAddon** können `syncCallbacks` verwendet werden, um synchronisierte Funktionen effizienter in Skripten einzusetzen.

* Erweiterung von Commands: Das Script fügt neue, benutzerdefinierte Befehle hinzu, die auf die Anforderungen des EWX-Frameworks zugeschnitten sind.

* Optimierte Server-Kommunikation: **esxAddon** erleichtert die Kommunikation zwischen Server und Client für eine verbesserte Steuerung und Synchronisierung.

* Anpassungsfähige Funktionen für Entwickler: Entwickler erhalten durch **esxAddon** flexible Funktionen zur Anpassung und Integration neuer Features.

* Kompatibilität und Erweiterbarkeit: **esxAddon** ist vollständig in das EWX-Framework integriert und bleibt mit zukünftigen Updates kompatibel.
